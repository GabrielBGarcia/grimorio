# Grimório — Ficha de RPG

Projeto web de ficha de RPG medieval dark/minimalista.

## Como executar

1. Extraia o arquivo ZIP
2. Abra o arquivo `index.html` no navegador

## Recursos
- Sistema de HP
- Inventário
- Magias
- Talentos
- Homebrew manager
- Upload de retratos
- Exportação JSON/ZIP

## Estrutura
- `index.html` → aplicação principal
